# 🃏 DOMIJACK - El Mejor Blackjack de RD

**DOMIJACK** es una implementación clásica del juego de cartas Blackjack (21) desarrollada en **C#** para la consola. El proyecto simula la experiencia de un casino con reglas auténticas, incluyendo la gestión dinámica del valor del As y la lógica automatizada del Dealer.

## 🚀 Características

* **Lógica de Juego Real:** El Dealer está obligado a pedir carta con 16 o menos y plantarse con 17 o más.
* **Gestión de Ases:** Los Ases cambian su valor de 11 a 1 automáticamente si el jugador supera los 21 puntos.
* **Interfaz Visual en Consola:** Uso de arte ASCII, colores para diferenciar palos (corazones/diamantes en rojo) y una mesa de juego visualmente organizada.
* **Barajado Aleatorio:** Implementación del algoritmo de intercambio para asegurar que cada partida sea única.
* **Tutorial Integrado:** Sección de reglas dentro de la aplicación para nuevos jugadores.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** C# (.NET Core / .NET Framework)
* **Entorno:** Console Application

## 🎮 Cómo Jugar

1.  **Nueva Partida:** Inicia un juego donde recibes dos cartas y el Dealer oculta una.
2.  **Pedir (P):** Solicita una carta adicional para acercarte a 21.
3.  **Quedarse (Q):** Termina tu turno y deja que el Dealer juegue su mano.
4.  **Objetivo:** Sumar 21 o tener una puntuación mayor que el Dealer sin pasarte.
